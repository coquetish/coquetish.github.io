|浏览量:

一、代码风格的格式化

1.在运算符两侧添加空格

int sum = a + b;

2.不同功能的语句单独一行

int a = 10;
int b = 20;
int sum = a + b;

3.控制结构的块的标准格式

if (ok) {
    // 代码块
} else {
    // 其他代码块
}

4.长表达式

int a[3][3] = {1, 2, 3,
               4, 5, 6,
               7, 8, 9
};

5.在除末尾的表点符号后面敲击空格

for (int i = 0; i < 114514; ++i) {
        printf("114514");
}

6.在关键字（如if、for、while）和后面的括号之间添加空格

7.代码的缩进，一般按一下Tab键即可

for (int i = 0; i < 114514; ++i) {
    int j = i;
    int ans = 0;
    while (j) {
        ans += j %= 10;
        j /= 10;
    }
    if (ans > 10) {
        printf("%d\n", ans);
    } else {
        printf("QAQ\n");
    }
}

常规的代码写法

缩进做好看着也挺好

#include <stdio.h>
int main(void){
    int n;scanf("%d",&n);
    for(int i=0;i<n;++i){
        int x;scanf("%d",&x);
        printf("%d%c",x,"\n"[i==n]);
    }
    int a=3,b=1,c;
    c=a+b;
    printf("%d\n",c);
    return 0;
}

官网文档的格式化写法

个人推荐第二种代码风格的写法（虽然开始会不习惯， 但是后面会很舒服）

#include <stdio.h>
int main(void) {
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        int x;
        scanf("%d", &x);
        printf("%d%c", x, " \n"[i == n - 1]);
    }
    int a = 3, b = 1, c;
    c = a + b;
    printf("%d\n", c);
    return 0;
}
