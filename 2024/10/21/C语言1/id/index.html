三目运算符(在书上P39，又叫条件表达式)

三目用的好可以简单替换一些但分支的if else，让代码更加简洁
表达式1 ？ 表达式2 ：表达式3

#include <stdio.h>
int main(void) {
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        printf("%d%c", i, i == n - 1 ? '\n' : ' ');
    }
    return 0;
}

字符数组的访问

结合字符数组和表达式的真值一种方法
这是一种字符串索引的灵活引用，这个字符串为” \n”，空格+换行，然后通过后面的表达式的真值来表示下表，1为真，即返回字符串的1号字符，换行符 ‘\n’, 0为假，返回字符串的0号字符，空格 ‘ ‘

#include <stdio.h>
int main(void) {
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; ++i) {
        printf("%d%c", i, " \n"[i == n - 1]);
    }
    return 0;
}
